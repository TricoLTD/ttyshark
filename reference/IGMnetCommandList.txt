**New Unit Instructions:
Run 7E command to determine unit type
Run 7F command to determine buffer sizes

   If Unit returns A= S= C=, then
           Alarms Address    = A *BF command get alarm values
           Data Address      = S
           Controls Address  = C  *If required else = 0
           Run Hours Address = Run B2 command to get size if required else = 0

   If Unit returns E= S= C=, Alarm Buffer size is variable then
           Alarms Address    = 2 *BC command get alarm values
           Data Address      = S
           Controls Address  = C  *If required else = 0
           Run Hours Address = Run B2 command to get size if required else = 0
